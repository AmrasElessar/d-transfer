# JetBrains Mono fontları

Bu klasöre **JetBrains Mono** woff2 dosyaları manuel olarak düşürülmelidir.

İndirme: https://www.jetbrains.com/lp/mono/ (sayfa altındaki "Download" linki)

Beklenen dosyalar:

```
public/fonts/
├── JetBrainsMono-Regular.woff2
├── JetBrainsMono-Medium.woff2
└── JetBrainsMono-Bold.woff2
```

JetBrains Mono **SIL Open Font License 1.1** altındadır — DTransfer'ın
GPL-3.0-or-later lisansından bağımsız olarak font lisans şartlarına tabidir
(redistribution kuralları için kendi lisansını oku).

Bu dosyalar repo'ya commit'lenebilir (binary dosya boyutu makul: 3 dosya × ~30 KB).
